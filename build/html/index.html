Unieke website op maat gemaakt

Webdesign is een vorm van toegepaste kunst dat in dienst staat van de gebruikservaring van de bezoeker: de essentie van de website hoort voor de bezoeker z.s.m. ondubbelzinnig duidelijk te zijn.

Vragen die door het ontwerp beantwoord zullen moeten worden:

  • Voor wie is de desbetreffende website bedoelt?
  • Welke boodschap moet duidelijk overkomen?
  • Wat betekent deze boodschap voor de visuele hiërarchie van de elementen in een webpagina?
  • Hoe stelt het ontwerp “het oog” in staat om deze lijn te volgen?

Vragen ook die door de getrouwe uitwerking van het ontwerp – web-bouw (web-development) – beantwoord zullen moeten worden. Want web-bouw is een ambacht met als elementair gereedschap HTML, CSS en JavaScript. Atelier Bram de Haan begint bij het bouwen van een website iedere keer weer met een leeg canvas: ik stel eer in een semantische HTML-markup met effectieve stijlen, z.v.m. gebruikmakend van “vanilla javaScript”, en zo weinig mogelijk gebruikmakend van zgn. “frameworks”. Een website optimaliseren zodat het snel laadt – is een sport die vanaf het allereerste begin van web-bouw in het achterhoofd zit, want een goed presterende snelle website is een essentiële kwaliteit voor de gebruikservaring.
